Handoff Loop-back Gate — the phase-gate a pass must clear; number = items in the Phase Gate Checklists
A record moves on readiness, not the calendar — each gate is a bar the whole building can see. Leads trigger → Creative builds & QCs → Audience Dev + Support activate in parallel → Campaign Ops confirms readiness & schedules. Counts map to the building-wide Phase Gate Checklists (184 items, five gates); a failed QC loops back to revision.
