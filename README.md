## To update a file

1. Replace the file in this repo
2. Commit and push
3. GitHub Pages auto-deploys in ~60 seconds
4. Same URL, new content — no link changes needed

## To add a new meeting doc

1. Add the HTML file to `meetings/`
2. Update `index.html` to include the new link
3. Commit and push
