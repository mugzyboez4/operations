How the work moves — step by step

The same spine, unpacked. Three lanes show exactly how a project moves through Creative, Design and QC — from brief in to archive.

Creative ManagerHow a project moves
01
Brief In
Take the ticket + campaign brief; confirm deliverables, budget envelope and timeline with the Campaign Lead.
02
Concept
Develop the treatment; pull moodboards with Design; align to creative direction before spend.
03
Resource
Assign the internal team or commission talent — directors, stylists; book vendors, raise POs.
04
Produce
Run the shoot / production; own logistics and the on-the-day calls.
05
Review
Edit + internal rounds; manage the artist / management feedback loop to lock.
06
QC + Deliver
Hand to QC for specs and ingestion; route the final files out.
07
Wrap
Close POs and invoices; file finals to Asset HQ / DAM.
Music video is the signature version of this flow: greenlight → director + treatment → pre-pro → shoot → post → QC + delivery → social cutdowns (Motion) + archive.
DesignHow design moves
01
Brief In
Take the design ticket — references, specs, deadline; clarify what's actually needed.
02
Explore
Direction options + moodboards; check against brand identity and the campaign look.
03
Design
Build artwork, packaging, logos, banners, ad units; keep versions clean.
04
Review
Design-lead review, then rounds with the Creative Manager, artist and management.
05
Finalize
Prep delivery specs and export formats; lock packaging specs.
06
QC + Route
QC check, route to delivery, file source + finals to the DAM.
Production / QCThe last mile — QC & routing (Chris S)
01
Intake
Take the finals + the ticket; confirm what's delivered and to which specs / platforms.
02
Spec Check
Check against delivery specs and platform / DSP requirements; flag anything off.
03
QC
CI QC pass — technical quality, versions, metadata.
04
Fix Loop
Kick issues back to Creative Mgr / Design; confirm corrected files.
05
Ingest
Ingest approved assets into the system.
06
Route + Deliver
Route final files to their destinations — DSP, partners, socials, physical.
07
Archive
File source + finals to Asset HQ / DAM — tagged and findable.
