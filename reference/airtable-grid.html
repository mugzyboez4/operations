📖 Glossary

DDEX
Digital Data Exchange — the global standard for how music metadata is structured and shared between labels, distributors, and DSPs.
ERN
Electronic Release Notification — the DDEX message used to deliver releases to DSPs. Four required sections: PartyList (credits), ResourceList (masters/artwork), DealList (pricing/rights), ReleaseList (title/genre/UPC). DSPs reject incomplete ERNs.
DSN
Delivery Start Notice — RCA's internal filing that initiates distribution. Unlike ERN, DSN accepts incomplete data (artwork TBD, blank rights). This is where the gap starts.
GRPS
Global Release Planning System — Sony's system-of-record for release planning and ISRC assignment. Feeds Release Runway but doesn't provide cross-department visibility.
Runway
RCA's real-time release schedule. Tracks 18 fields per release — 7 are chronically "Unknown" because the data originates in departments that don't update Runway.
SAA
Single Asset Authorization — required before distribution. Needs sign-off from both labels' Business & Legal teams. Shows "Unknown" in Runway.
SAMP
Sample Clearance — confirms all sampled material is legally cleared. Tracked in email and legal files, not in any shared system.
CARMA
Sony's Global Contract Management System — houses artist contracts, deal terms, territory rights, and option dates.
