Glossary

DDEX
Digital Data Exchange — the global standard for how music metadata is structured and shared between labels, distributors, and DSPs.
ERN
Electronic Release Notification — the DDEX message for delivering releases to DSPs. Four sections: PartyList, ResourceList, DealList, ReleaseList. DSPs reject incomplete ERNs.
DSN
Delivery Start Notice — RCA's internal filing that initiates distribution. Unlike ERN, DSN accepts incomplete data. This is where the gap starts.
GRPS
Global Release Planning System — Sony's system-of-record for release planning and ISRC assignment.
Runway
RCA's real-time release schedule. Tracks 18 fields per release — 7 are chronically "Unknown."
SAA
Single Asset Authorization — required before distribution. Needs both labels' Legal sign-off.
SAMP
Sample Clearance — confirms sampled material is legally cleared. Tracked in email, not any shared system.
CARMA
Sony's Global Contract Management System — artist contracts, deal terms, territory rights.
