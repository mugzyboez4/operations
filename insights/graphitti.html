Section 02 · Now

Three priorities

Sequenced, not parallel. Cleanup ships in days. Data scaling unblocks accuracy at scale. Discovery Dashboard is a scoping milestone — no build until scope is locked.

Lane 01
Cleanup & refinement
Surfaced from A&R user feedback post-rollout. Most are quick fixes; sequencing handled at the build level.
Desktop
  • Two master lists — Frontline and Trend Senate aggregates
  • "Add all" button — restore for markets and songs
  • % of total per market — on Ex-US and US Cities pages
  • "Albums Released" — remove (data unreliable, e.g. SZA shows 35)
  • "US" → "Artist" — on artist page, matches Songs / Albums framing
  • Songs page legend — replace with per-line tooltip on graph
  • Weekly aggregation toggle — match the mobile version
  • % chg behavior — should release when scrolling line, not stay fixed
  • Average streams dotted line — remove
  • "US Cities" → "US States" or "US Map"
Mobile
  • Spark lines — fix Ex-US instability
  • % change column — add to artists and tracks tables
  • Genre split — fix Rap separating from R&B/Hip-Hop; should be R&B and Rap/Hip-Hop
  • Axis labels — add x and y, match desktop
Lane 02
Data scaling & profile merging
Accuracy work that unblocks broader rollout to leadership. Subsumes several items that were "in progress" in March.
  • Artist profile merging — collapse duplicates, fix data gaps across sources
  • Global data — Ex-US and Global incorporation across the platform
  • Genre tagging — aggregate best metadata from Luminate, Chartmetric, Spotify
  • Label mapping — accurate label and distributor information
  • Financial label alignment — reconcile to corporate definition before broader leadership rollout
Lane 03 · Scoping
Discovery Dashboard
Strategic priority. A low-lift surface for unsigned talent so we don't miss the big ones. Net-new since the March roadmap.
Scope first, build second. Phase 01 defines what gets built. Phase 02 starts only when scope is locked.
Phase 01 · Scope
Define before build
  • User — A&R only, or does it surface to leadership?
  • Input signals — which Graphitti data points trigger inclusion?
  • Surface — one dashboard, multiple per genre, mobile-first?
  • Success metric — saves vs. signs, leads generated, hit rate?
